# Семантический поиск настольных игр 🎲

## О проекте

Это веб-приложение, которое помогает подобрать настольную игру по **смыслу**, а не только по тегам.

Вы просто описываете словами, что хотите — и система находит подходящие игры из базы BoardGameGeek.

## Зачем это нужно?

Обычные фильтры ("игроков 2–4", "длительность до 60 минут") полезны, но они не понимают ваш реальный запрос. Например:

❌ Фильтры не ответят на вопросы вроде:
- *«Хочу что-то похожее на Каркассон, но более комплексное»*
- *«Нужна атмосферная игра про приключения и путешествие»*
- *«Ищу несложную экономическую игру с развитием производства»*

✅ Наша система **понимает смысл** таких запросов и рекомендует игры, которые действительно подходят.

## Как это работает (простыми словами)

1. **База игр** — данные о настольных играх загружаются из BoardGameGeek (названия, описания, механики, категории, изображения).
2. **Семантический поиск** — система переводит игру и ваш запрос в «смысловые векторы» и ищет ближайшие по смыслу.
3. **Фильтрация** — к результатам применяются ограничения по числу игроков, длительности и сложности.
4. **Объяснение** — к каждой рекомендации показывается, почему она подходит (совпали механики, похожа тема, подходит по составу игроков и т.д.).

## Что умеет система

- 🔍 Поиск по свободному тексту на естественном языке
- 🎯 Опциональная игра-ориентир («похоже на ...»)
- 👥 Фильтры по числу игроков
- ⏱️ Фильтры по длительности партии
- 📊 Фильтры по сложности
- 🖼️ Карточки игр с изображениями
- 💡 Краткие объяснения рекомендаций

## Что входит в MVP

- Загрузка игр из BoardGameGeek
- Хранение данных в PostgreSQL
- Семантический поиск
- Простой веб-интерфейс
- Выдача списка похожих игр с объяснениями

Это осознанное упрощение: проект сосредоточен именно на **семантическом поиске**, а не на полноценном магазине или социальной сети.

## Технологии (кратко)

- Backend: Python + FastAPI
- База данных: PostgreSQL + pgvector (хранение векторных эмбеддингов)
- Семантика: sentence-transformers (all-MiniLM-L6-v2)
- Данные: BoardGameGeek XML API

## Статус проекта

Проект находится в разработке. Это некоммерческий пет-проект для демонстрации и портфолио.
